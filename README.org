* dot-cons-tree

  This is a cons tree visualiser that uses graphviz.

** Author

   Jacek Podkanski

** Licence

   Public domain

** REPL use

   Assuming the project is in your home folder you can run the following in your
   REPL.

   #+begin_src lisp
     (push #p"~/dot-cons-tree/" asdf:*central-registry*)
     (ql:quickload :dot-cons-tree)
   #+end_src
